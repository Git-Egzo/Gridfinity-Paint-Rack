Privacy Policy – Gridfinity Paint Rack for Autodesk Fusion

Effective date: March 1, 2026

1. Data Collection

Gridfinity Paint Rack does not collect, store, transmit, or process any personal data. The plugin operates entirely within the local Autodesk Fusion environment and writes only a diagnostic log file (paintRack_debug.log) to the user's Desktop. This file contains no personal information — only technical model parameters. It can be deleted at any time by the user.

2. No Third-Party Data Sharing

No data is shared with any third party. The plugin does not use analytics SDKs, crash-reporting services, or external APIs.

3. Autodesk Data

Autodesk Fusion and the Autodesk App Store may collect data in accordance with Autodesk's Privacy Statement. The author of this plugin has no control over Autodesk's data practices.

4. User Rights

Since no personal data is collected, there is no data to access, correct, or delete. For questions, contact: TWÓJ EMAIL
